## 📝 Note Tecniche

### Prisma Relations
```typescript
// Client → Dentists (one-to-many)
client Client @relation(fields: [clientId], references: [id], onDelete: Cascade)

// Case → Dentist (many-to-one, optional)
dentist Dentist? @relation(fields: [dentistId], references: [id])
```

### API Patterns
- **GET** `/api/dentists/client/:clientId` - Filtra per cliente
- **POST/PUT** - Usa JSON body
- **DELETE** - Cascade su dentisti quando elimini cliente

### Frontend State Management
- useState per form data locale
- useEffect per loading dati iniziali
- fetch API diretta per dentisti (non servizio centralizzato)
