🧹 Pulisci Cache Browser

Perché usare questo tool?

Se riscontri problemi con il portale clienti (upload file, creazione casi, errori API), clicca il pulsante sotto per pulire completamente la cache del browser.

Questo tool pulirà:

  • localStorage (incluso auth-storage)
  • sessionStorage
  • IndexedDB
  • Cookies dell'applicazione
  • Service Workers cache
✅ Cache pulita! La pagina si ricaricherà automaticamente...
